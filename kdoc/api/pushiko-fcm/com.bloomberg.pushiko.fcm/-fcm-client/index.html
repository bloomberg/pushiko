FcmClient

@ThreadSafe
class FcmClient

A client for sending push notification messages to Firebase Cloud Messaging (FCM).

Clients internally use the shared HTTP connection pool for connecting to FCM. They are inherently thread-safe and normally intended to live until the JVM begins shutting down. Clients gracefully close in an orderly manner, releasing their connections resources and closing their session. Once closed, clients can no longer be used to send any notification requests.
