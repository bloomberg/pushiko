ApnsClient

@ThreadSafe
class ApnsClient

A client for sending push notification messages to Apple Push Notifications service (APNs).

Clients internally use an HTTP connection pool for connecting to APNs. They are inherently thread-safe and normally intended to live until the JVM begins shutting down. Clients gracefully close in an orderly manner, releasing their connections resources. Once closed, clients can no longer be used to send any notification requests.
