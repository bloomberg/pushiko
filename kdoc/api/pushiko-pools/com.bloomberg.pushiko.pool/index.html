@ThreadSafe
class CommonMuxPool<R : Any, P : Poolable<R>>(configuration: PoolConfiguration, factory: Factory<P>, recycler: Recycler<R>) : SuspendPool<R, P>

A non-blocking, lock-free pool that maintains a minimum number of pooled multiplexers even in the absence of demand, heuristically scheduling the creation of new objects if need be and if the headroom exists to do so even when there may be capacity available elsewhere. When no further aggregate capacity is available further acquisition attempts are treated as pending and their associated continuations buffer up over the pool, only resuming when capacity becomes available again. If there are too many queued pending acquisitions the pool begins failing the oldest of these by resuming the associated continuation with an exception.
