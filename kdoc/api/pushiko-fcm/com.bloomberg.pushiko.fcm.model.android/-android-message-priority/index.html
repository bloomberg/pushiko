AndroidMessagePriority

@Serializable
enum AndroidMessagePriority : Enum<AndroidMessagePriority>

Priority of a message to send to Android devices. Note this priority is an FCM concept that controls when the message is delivered. Additionally, you can determine notification display priority on targeted Android devices using AndroidNotificationWriter.notificationPriority.
